Project News & Updates

Version 2.0: The Flexbox Overhaul

Posted on: April 10, 2026

We have officially transitioned our entire layout system from traditional floats to CSS Flexbox. This change allows for better alignment, equal-height columns, and a much smoother experience on mobile devices. The high-contrast "Brutalist" design remains, but the underlying code is now leaner and faster.

Security Patch: Form Validation

Posted on: April 5, 2026

We've updated our formCheck() JavaScript logic to improve client-side validation. New security measures ensure that passwords meet modern complexity requirements, and we've refined the gender selection interface for better accessibility.
